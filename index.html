p5.js Drawing Extended Example

This project is an example based on the p5.js continuous lines tutorial, enhanced with a color-changing feature.

The color change button was inspired by the W3Schools JavaScript event tutorial (https://www.w3schools.com/howto/howto_js_toggle_hide_show.asp).

Website Example by Ayoung Chun. Used tutorials : p5.js continuous lines, W3Schools JS.
